Tipos de selectores que entran en examen

Clase (class=””)

Clase es un selector que nos permite seleccionar varios elementos html que tengan la misma clase.

Id (id=””)

Id es un selector que nos permite seleccionar un solo elemento html que tenga un id especifico.

Todos los elementos (*)

Todos los elementos es un selector que nos permite seleccionar todos los elementos html.

Todos los elementos contenidos en un elemento ( p, em, h1, h2)

Todos los elementos contenidos en un elemento es un selector que nos permite seleccionar todos los elementos html que esten dentro de un elemento html.

Elementos contenidos en otros elementos (ol li)

Elementos contenidos en otros elementos es un selector que nos permite seleccionar todos los elementos html que esten dentro de otros elementos html.

Sobrevuleo

Sobrevuleo es un selector que nos permite seleccionar un elemento html que este sobrevuleando otro elemento html.
